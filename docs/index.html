Relay

Async by default — no await keyword
Python-like syntax • Built-in web server & HTTP client

GitHub Quick Start

Async without await

sleep(2000, print("world"))
print("hello")

→ prints "hello" right away, "world" after 2 seconds

Simple Web Server

app = WebApp()
server = WebServer()

@app.get("/")
fn index()
    return "hello world"

server.run(app)

git clone https://github.com/patx/relay-lang
cd relay-lang
cargo install --path .

relay test.ry
